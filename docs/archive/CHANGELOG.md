## 📋 版本说明

### 版本号规则

采用 [语义化版本](https://semver.org/lang/zh-CN/) 规范：

- **主版本号（X）**：不兼容的 API 修改
- **次版本号（Y）**：向下兼容的功能性新增
- **修订号（Z）**：向下兼容的问题修正

### 变更类型

- ✨ **新增**（Feature）：新功能
- 🐛 **修复**（Fix）：bug 修复
- 📚 **文档**（Docs）：文档更新
- 🎨 **样式**（Style）：代码格式（不影响功能）
- ♻️ **重构**（Refactor）：代码重构
- ⚡ **性能**（Perf）：性能优化
- ✅ **测试**（Test）：测试相关
- 🔧 **构建**（Build）：构建工具
- 🚀 **部署**（Deploy）：部署相关
- 🔒 **安全**（Security）：安全相关
